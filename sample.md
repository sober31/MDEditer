# MDEditer 示例文档

欢迎使用 **MDEditer** —— 一款好用的 Markdown 查看 / 编辑工具。

## 功能一览
- [x] 源码 / 预览 / 分屏 三种视图
- [x] 点击勾选任务清单（自动同步源码）
- [ ] 打开本地 .md 文件（拖拽或按钮）
- [ ] 一键保存 / 另存为（Chrome/Edge 直接写回原文件）
- [ ] 明暗主题、自动草稿、文档大纲、查找高亮

## 文本样式
支持 *斜体*、**加粗**、~~删除线~~、`行内代码` 与[链接](https://www.codebuddy.cn)。

> 引用块：工具链越顺手，写作越顺畅。

```js
function hello() {
  console.log('Hello MDEditer');
}
```
